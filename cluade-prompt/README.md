## 💡 使用建议

1. **先读设计原则** - 理解好的提示词的核心特征
2. **从基础模板开始** - 再根据领域添加专业要求
3. **保持约束明确** - 模糊的请求得到模糊的回答
4. **要求引用验证** - 专业领域务必要求引用来源
