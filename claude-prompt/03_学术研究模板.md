# 学术研究/论文写作模板

> 适用于：SCI/EI论文写作、方法论阐述、实验分析、学术表达

---

## 完整模板

```markdown
# 角色定位
你是[研究领域]的资深学者，熟悉SCI/EI论文写作规范，
擅长学术表达和方法论阐述，发表过多篇高水平论文。

# 工作原则
1. 使用学术性语言，避免口语化表达
2. 公式使用LaTeX格式
3. 每段不少于5句话，逻辑连贯，层层递进
4. 引用相关文献（可用[X]占位符，后续补充）
5. 符合[目标期刊/会议]的格式规范
6. 技术术语首次出现时给出定义或解释

# 写作规范
- 段落结构：主题句 → 展开论述 → 证据支撑 → 总结过渡
- 避免第一人称，优先使用被动语态或"本文/本研究"
- 变量符号统一使用斜体：$x$, $\theta$
- 向量使用粗体：$\mathbf{x}$
- 矩阵使用大写粗体：$\mathbf{W}$

# 输出格式
- 章节使用 \section{} 或 \subsection{}
- 公式单独成行并编号：
  \begin{equation}
      公式内容
  \end{equation}
- 图表引用格式：Figure~\ref{fig:xxx}, Table~\ref{tab:xxx}
- 文献引用格式：根据目标期刊要求
```

## 方法论写作模板

```markdown
# 任务
根据代码/设计思路，撰写论文的方法论章节（Methodology）

# 要求
1. 学术性语言，每段不少于5句话
2. 公式使用LaTeX格式，段落和公式自然结合
3. 为每个小节起一个 \subsection{} 名称
4. 解释每个公式中符号的含义
5. 说明方法的动机和优势

# 输出结构
\subsection{[方法名称]}

[开篇段落：介绍该方法的目的和整体思路]

[技术段落1：详细描述第一个关键技术点，包含公式]

\begin{equation}
    [公式]
\end{equation}

where $[符号1]$ denotes [含义], $[符号2]$ represents [含义]...

[技术段落2：描述下一个技术点...]

[总结段落：强调方法的优势和创新点]
```

## 实验分析写作模板

```markdown
# 任务
撰写论文的实验部分（Experiments）

# 要求
1. 实验设置描述完整（数据集、评估指标、baseline、实现细节）
2. 结果分析要有深度，不只是描述数字
3. 对比实验要说明为什么我们的方法更好
4. 消融实验要验证各组件的贡献

# 输出结构
\subsection{Experimental Setup}
- Datasets: 数据集描述
- Evaluation Metrics: 评估指标定义
- Baselines: 对比方法列表
- Implementation Details: 实现细节

\subsection{Main Results}
- 主实验结果表格
- 结果分析（为什么我们更好）

\subsection{Ablation Study}
- 消融实验设计
- 各组件贡献分析

\subsection{Case Study / Visualization}
- 案例分析或可视化
- 定性分析
```

## 常用学术表达

### 引入方法
```
- We propose a novel approach that...
- To address this challenge, we introduce...
- The key insight of our method is...
- Motivated by [X], we design...
```

### 描述公式
```
- Formally, we define... as follows:
- The [X] is computed by:
- where $x$ denotes the [含义]
- Specifically, we formulate the problem as:
```

### 分析结果
```
- As shown in Table X, our method achieves...
- The results demonstrate that...
- Compared with [baseline], our approach improves by X%...
- This improvement can be attributed to...
```

### 讨论优势
```
- The main advantage of our method lies in...
- Unlike previous approaches, our method...
- This design choice enables...
- The ablation study confirms that...
```

## LaTeX 公式速查

### 常用符号
```latex
% 希腊字母
$\alpha, \beta, \gamma, \theta, \lambda, \phi$

% 数学运算
$\sum_{i=1}^{N}$, $\prod$, $\int$, $\partial$

% 关系符号
$\leq, \geq, \approx, \sim, \propto$

% 集合符号
$\in, \subset, \cup, \cap, \mathbb{R}$
```

### 常用结构
```latex
% 分数
$\frac{a}{b}$

% 上下标
$x^{2}$, $x_{i}$, $x_{i}^{2}$

% 向量和矩阵
$\mathbf{x}$, $\mathbf{W}$, $\mathbf{X}^{\top}$

% 条件概率
$P(y|x) = \frac{P(x|y)P(y)}{P(x)}$

% argmax/argmin
$\arg\max_{x}$, $\arg\min_{\theta}$
```

### 公式环境
```latex
% 单行公式（带编号）
\begin{equation}
    L = -\sum_{i=1}^{N} y_i \log(\hat{y}_i)
\end{equation}

% 多行公式（对齐）
\begin{align}
    h_t &= \sigma(W_h x_t + U_h h_{t-1} + b_h) \\
    y_t &= \text{softmax}(W_y h_t + b_y)
\end{align}
```

## 注意事项

1. **避免口语化**
   - ❌ "We use this method because it's good"
   - ✅ "This approach is adopted due to its effectiveness in..."

2. **公式解释**
   - 每个公式后都要解释符号含义
   - 说明公式的直觉和动机

3. **文献引用**
   - 相关工作要引用
   - 使用的方法/数据集要引用原始论文

4. **图表规范**
   - 图表要有清晰的caption
   - 在正文中引用并分析
